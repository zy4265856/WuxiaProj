# 🗡️ 武侠项目 Obsidian 知识库 - 插件安装指南

## 安装路径

- [ ] Obsidian 已安装 (v1.12.7)
- [ ] Vault 已初始化为项目根目录 `D:\Proj\WuxiaProj`
- [ ] 核心插件已启用
- [ ] 模板文件已创建

## 📦 推荐安装的社区插件

打开 Obsidian → 设置 → 第三方插件 → 关闭安全模式 → 浏览社区插件市场，逐一安装：

### 必备 (强烈推荐)

| 插件 | 用途 | 搜索关键词 |
|------|------|-----------|
| **Dataview** | 将笔记当作数据库查询，自动生成列表/表格 | `dataview` |
| **Kanban** | 看板任务管理 | `kanban` |
| **Excalidraw** | 手绘风格图表、流程图、UI草图 | `excalidraw` |
| **Templater** | 高级模板引擎，比内置模板更强大 | `templater` |

### 推荐

| 插件 | 用途 | 搜索关键词 |
|------|------|-----------|
| **Calendar** | 日历视图，快速跳转每日笔记 | `calendar` |
| **Obsidian Git** | Git 版本控制集成，自动备份 | `obsidian git` |
| **Admonition** | 提示框/警告框 Callout 美化 | `admonition` |
| **Outliner** | 增强大纲/列表编辑体验 | `outliner` |
| **Table Editor** | 表格编辑器，像 Excel 一样编辑 | `table editor` |

### 可选

| 插件 | 用途 | 搜索关键词 |
|------|------|-----------|
| **Mind Map** | 思维导图 | `mind map` |
| **Style Settings** | 主题样式微调 | `style settings` |
| **Icon Folder** | 文件夹图标美化 | `icon folder` |

## 🚀 如何使用

1. 打开 **Obsidian**
2. 点击「打开本地仓库」→ 选择 `D:\Proj\WuxiaProj` 文件夹
3. 首次打开时，关闭「安全模式」以启用社区插件
4. 在社区插件市场搜索安装上述插件
