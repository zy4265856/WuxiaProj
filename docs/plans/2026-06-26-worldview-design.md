# 武侠世界观总纲

## 〇、文档定位与范围

本文档是 WuxiaProj 的世界观「宪法」——锁定所有下游创作（武功设计、美术、NPC、势力、剧情、关卡）都必须遵守的顶层设定与基调。它不细化任何单一系统，而是为后续各详设文档提供统一的设定地基与北极星。

**在范围内**：武力层级、时代背景、世界规模与地理骨架、核心冲突前提、玩家定位、主题母题、魔幻/超自然边界、势力骨架与格局、命名·美术·世界规则基调。

**不在范围内（指向后续子 spec）**：

| 内容 | 归属（待创建） |
|------|----------------|
| 各门派名号 / 人物 / 武功 / 关系网 | `YYYY-MM-DD-factions-detail.md` |
| 各区域身份 / 地名 / 连接拓扑 / 地图 | `YYYY-MM-DD-geography-detail.md` |
| 武学体系详设（内外家 / 流派 / 神功传说） | `YYYY-MM-DD-martial-arts-system.md` |
| 主线剧情节点 / 关键 NPC / 奇遇 | `YYYY-MM-DD-main-story.md` |
| 斩首计划细节、朝廷鹰犬组织 | 归入主线剧情或势力详设 |

## 一、世界基调：中武 · 偏写实

- **武力金字塔**：99% 武者能力在常理之内；唯**金字塔顶尖的传说级武者**可展现超出常识的能力（仍是武学范畴、凡人尺度，**非**御剑 / 法术 / 妖魔）。
- **内功内息是真实战力**：掌力隔空、护体真气、点穴封脉、轻功跃檐——与现有 [属性系统设计](2026-06-25-attribute-system-design.md) 的外家(气劲)/内家(内息)分流一致。
- **神功传说**真实存在，作为**稀有高端内容**（养成目标 / 奇遇回报），不日常化。
- **战棋一致性**：保留凡人尺度，顶尖奇观不破坏网格战棋「走位＝战术核心」的设计。

---

## 二、时代背景：架空 · 乱世式微

- **架空**：虚构朝代，借晚唐 / 元末 / 明末式**质感锚点**（衣冠器物、官制、地名风格），**不绑真实历史年表与人物**。
- **乱世式微**：中央权威衰落、割据蜂起、帮会流民横行、江湖与庙堂激烈交锋——「侠以武犯禁」最有张力的舞台。
- 选乱世而非盛世：不稳定的世界才值得游历冒险，并天然催生多方博弈。

---

## 三、世界规模与地理骨架：式微旧帝国 ＋ 5 大区域

- **结构**：一个式微旧帝国（名义有朝廷 / 旧都但失控）＋ 数个自治 / 割据大区域——**中心（庙堂 / 旧都）＋ 边缘（江湖 / 四方）双层结构**。
- **规模**：目标 **5 个大区域**＝1 个中原腹地 / 旧都圈（繁华、腐败、权力漩涡）＋ 3~4 个边陲区域（山岳 / 水乡 / 大漠 / 密林…，各由地方门派主导）。
- **玩家起步**：初始落其中一个区域，逐步解锁；MVP 可先聚焦 1~2 个做深。
- 朝廷力量集中于中原旧都圈，向边陲辐射渗透。
- 具体区域身份 / 地名 / 连接拓扑 → 《地理详设》。

---

## 四、核心冲突：朝廷双线 vs 门派割据

乱世之下，朝廷为重建集权、从门派手中夺回**地域资源与人口的控制权**，走双线：

- **明面**：以朝廷大义对各门派**长臂管辖**、处处掣肘，并**挑拨各派关系**令其内耗；
- **暗里**：推进惊天**「斩首计划」**——意图一举覆灭各派**高层战力**（＝武林权力结构）。

→ **「斩首计划」即主线脊柱**，自带倒计时与高潮。门派是中央集权的最大障碍；江湖与庙堂的冲突被推到生死存亡的高度。

---

## 五、玩家定位：穿越者 · 外门弟子（纯叙事）

- **身份**：某江湖门派**外门弟子**，本拟于**入宗大典**正式拜门；
- **起点事件**：大典遭**放毒偷袭**，原主中毒将死——这正是**朝廷嫁祸敌对宗门**的一环（斩首计划前奏 / 试探）；
- **穿越**：一名**具现代思维**的灵魂穿越至此、取代原主而康复；在仿古架空的中武世界闯出自己的天地。
- **机械度（重要）**：穿越身份＝**纯叙事背景，零机械效果，不占天赋槽**。天赋系统照常是「创建时固有、有实际效果的能力」，与穿越身份分开。
- **「现代见解 / 非常规分支」是内容设计取向，非机械门槛**：事件 / 奇遇**本身就按「主角是穿越者」来写**（带时代外解法 / 跳出礼教与门派愚忠的选项）。没有「本土对照主体」，每条路 inherently 都是穿越者的路。
- **羁绊天然脆弱**：因入宗大典被毒、**未正式入门**，与门派关系可由玩家自定义（效忠 / 出走 / 挑拨皆可）——保开放世界自由度。
- 待定：「本土 / 非穿越者主角」可选模式——有趣但工作量巨大，**本阶段不拍板**。

---

## 六、主题母题

- **主母题**：**「超越时代的自由意志 vs 旧秩序桎梏」**——穿越者的真正特殊性是「不被这个时代绑架」（门规 / 朝廷 / 礼教 / 正邪二元对人的规训）。
- **辅母题 ③ 道德灰度 / 侠义的重新定义**：朝廷阴险、门派未必正义、穿越者可黑可白——「侠」由玩家自定义。支撑「按己意抉择」的玩法。
- **辅母题 ④ 知见即力量 / 信息差驱动**：穿越者真正的底牌是「看穿」（识破挑拨、嫁祸、斩首计划）；信息差比蛮力更致命——呼应「见解超时代」与战棋的策略性。

---

## 七、魔幻 / 超自然边界：边缘超自然 · 武力金字塔

- **边缘超自然（可解读）**：允许**蛊术 / 奇毒 / 奇药 / 苗疆异术**有近乎神奇的效果，但**都归于毒理 / 药理 / 异术范畴，无真妖魔、无法术**。
- 严格排除：妖魔、鬼怪、法术、御剑飞行——与中武偏写实 + 「知见即力量」主题冲突。
- **武力金字塔**（见第一节）：仅顶尖传说级武者可超常识。
- 设计意图：保住「一切最终能被锐利心智拆解」，让穿越者的「看穿」优势不被真魔法稀释；同时给内容留诡异风味（毒 / 蛊 / 奇药 / 禁术异术）。

## 八、势力骨架与格局

### 势力分类（核心四类）

| 势力 | 定位 | 与主线关系 |
|------|------|-----------|
| **朝廷** | 中央集权回归的推动者；道德灰（有「平乱安民」一面，也有斩首阴谋） | 主线对手方；双线策略发起者 |
| **门派** | 地方割据主力，掌握地域资源 / 人口；武林主体 | 斩首计划目标；主角出身阵营 |
| **帮会** | 漕运 / 盐铁 / 黑道等地头蛇，世俗逐利，盘踞城镇商路 | 资源 / 人口 / 商路载体；可被朝廷或门派收买利用 |
| **散人 / 游侠** | 无门无派者（含主角早期） | 自由身份，不属任何势力 |

> 商会 / 宗教教派 / 异族暂不单列核心势力，需要时作为门派 / 帮会的变体或后续扩展（YAGNI）。

### 门派格局：模糊正邪（经济基础 → 行事风格 → 正邪观感）

- **不设干净的正邪二元**。正邪＝门派的**经济基础 / 行事纲领**，非标签：
  - 营收靠**青楼**等灰产 → 行事**邪魅**；
  - **道教系**自种自采自猎、收入板正 → 行事**质朴自然**；
  - **朝廷运营的门派**深嵌城镇治安却藏蝇营狗苟 → **灰色混沌**。
- **双层功能**：表层给玩家**游戏之初的兴趣指向**；里层**正中有邪、邪中有正**，让人物群像与故事立体丰满。
- 约 **5~8 家大门派**（每大区域 1~2 家主导），下设众多小门派与帮会。
- **武林盟 / 共主**：存在一个**名存实亡的武林盟**——正是朝廷「挑拨各派关系」与「斩首计划」的核心靶子；各派高层＝斩首目标，直接喂养主线。

### 门派－地域关系

- 每大区域由 **1~2 家地方门派主导**；大门派可跨区设分舵 / 据点。
- 朝廷力量集中于中原旧都圈，向边陲辐射渗透。

## 九、命名 · 美术 · 世界规则

### A. 命名基调
- **人名**：仿古写实汉名（金庸式，如「沈砚」「陆离」），**非**仙侠花哨名；
- **地名**：架空，质感锚晚唐 / 元末 / 明末；
- **武功 / 招式名**：传统武侠意象式，因中武偏写实而**偏刚劲朴拙、少仙气**。

### B. 美术基调
- 2D、中武偏写实、乱世式微；**沉郁而不黑暗，乱世里仍有侠气与烟火**。
- 关键词：沉郁苍凉、水墨 / 工笔质感、晚唐明末式衣冠器物、中心繁华腐朽 vs 边陲粗犷的视觉对比。

### C. 世界规则杂项
- 货币：银两 / 铜钱；度量：斤两 / 尺；纪年：架空**年号**＋ 干支；语言：通用语，**不造 conlang**。
- 原则：**借汉地通用器物 / 度量 / 纪年的外壳，朝代名 / 年号 / 地名架空自洽**——避免过度造世界给新手团队加负担。

## 十一、关键决策与理由（记录）

1. **中武偏写实**：既让内息 / 内伤系统名正言顺，又不破坏网格战棋的走位核心；留「神功传说」作稀有高端内容。
2. **架空而非真实朝代**：协作者（含新手）无需考据、不被史实卡死，可自由设计势力与事件。
3. **乱世式微**：不稳定的世界值得冒险，天然催生多方博弈，也是「侠以武犯禁」最有张力的舞台。
4. **5 大区域**：足够开放世界广度与章节感，又不让美术 / 关卡团队爆量。
5. **朝廷双线 + 斩首计划作主线脊柱**：自带倒计时与高潮，把「朝廷 vs 门派」落到可执行的剧情引擎。
6. **穿越者纯叙事零机械**：契合「道德自由」玩法，不破坏天赋系统，不稀释游戏机制纯净度。
7. **主母题①自由意志 + 辅③道德灰度 + ④知见即力量**：统领全局、去说教味、呼应穿越者特质与战棋策略性。
8. **边缘超自然 + 武力金字塔**：保诡异风味，又不让真魔法稀释「看穿」优势。
9. **模糊正邪＝经济基础 / 行事风格**：自洽涌现，完美服务道德灰度，给玩家初期兴趣指向 + 深入后的立体群像。
10. **借汉地外壳 + 架空年号地名**：降低造世界负担，团队友好。
