# 属性系统设计

## 〇、文档定位与范围

本文档定义角色的**属性系统**（十维属性及其作用方式）。

**核心立场**：属性**不进入骰池**。属性在系统中只承担两件事：

1. 作为**骰面公式的缩放输入**（数值调整）；
2. 作为**武功学习 / 装备使用的主属性门槛**。

> 这一立场修订了大纲原本"行动骰子池 = 基础属性骰 + 联动骰（基础属性骰由属性提供）"的设定。详见第 5 节「修订说明」。

**在范围内**：十维属性定义、缩放与门槛角色、门槛模型、成长占位、与现有文档的一致性修订。

**不在范围内（指向其他文档）**：

| 内容 | 归属 |
|------|------|
| 骰面伤害 / 各类公式的具体公式 | 骰面公式文档（待创建） |
| 属性成长曲线、数值范围、属性点获取 | 成长系统文档（待创建） |
| 才华系统（触发式、多渠道） | 才华系统文档（待创建） |
| 装备不达标的具体惩罚公式 | 装备系统文档（待创建） |
| 社交 / 探索结算框架 | 各自文档（待创建） |

## 一、核心模型：属性在系统中的位置

### 骰池构成（不含属性）

```
行动骰子池 = 基础骰 + 联动骰
```

- **骰池来源**：武功 + 装备 + 天赋 + 才华（+ 成长奖励的自定义骰面）。
- **保底约定**：空手且无任何武学时，角色仍持有一个预设的 **「挥击骰体」**（保底攻击骰），确保任何时候都可行动；无任何轻功时，角色持有一个预设的 **「步行」**（保底移动骰）

### 属性的两项职责

属性（共十维）**不产出骰子**，只做：

1. **缩放**：作为某些骰面公式的输入，缩放其数值（如气劲缩放外功伤害公式）。
2. **门槛**：作为武功学习（硬门槛）/ 装备使用（软惩罚）的判定依据。

> 关键约束：属性既不决定"有几枚骰、骰面是什么"，也不直接决定"打不打得中"（命中 / 未命中是骰面结果）。属性只调数值、卡门槛。

## 二、十维属性

每个属性**独占一条公式轴**，相互不重叠。

| 属性 | 缩放 / 修正（公式输入） |
|------|------------------------|
| **气劲** | 外功伤害 |
| **内息** | 内功伤害（内伤） |
| **根骨** | 气血上限、外伤防御、恢复 |
| **悟性** | 悟性类武学修正 |
| **身法** | 移动骰距离、闪避 |
| **定力** | 精神抗性、抗 debuff |
| **福缘** | 运气：掉落、奇遇、随机好运 |
| **魅力** | 社交骰面 |
| **精力** | 行动力阶梯（= 战斗文档行动力来源） |
| **精准** | 精准类武学修正 |

### 说明

- **气劲 / 内息**：是具体的伤害缩放轴。外家与内家流派在**属性层面即有差异**（外功走气劲、内功走内息），流派风格的其余差异由武功 / 装备 / 天赋体现。
- **悟性 / 精准**：只笼统描述为"X 类武学修正"，**不在此列具体效果**（如暴击、穿甲等）。其具体作用归**武功 / 公式文档**定义。
- **暴击 / 穿甲等不归属任何属性**：它们是武功公式的实现细节，由武功 / 公式文档规定。
- **主属性标签**：每门武功、每件装备、每项可学内容都带一个（或多个）"主属性"标签，由其主属性来卡门槛与缩放威力。

## 三、门槛与解锁模型

### 3.1 门槛严格度

| 内容 | 门槛类型 | 说明 |
|------|---------|------|
| **武功学习** | **硬门槛** | 达不到主属性要求则**无法学习** |
| **装备使用** | **软惩罚** | **随时可装备**；主属性不达标时受惩罚（效果打折 / 沿用战斗文档"装备超重惩罚"扣行动力上限的思路）。惩罚公式 → 装备系统文档 |

> 设计意图：装备走软惩，避免"捡到神装却用不了"的挫败；武功走硬门槛，保持养成目标感。

### 3.2 卡哪个属性（主属性门槛）

- 每门武功 / 每件装备有**一到多个"主属性"**，门槛由**其主属性**判定。
- **悟性不是通用学习门槛**：它只是"悟性类武功"的主属性，与其它属性行为一致。
- **多主属性要求**：允许。一门武功 / 一件装备可配置 1~ 多个主属性门槛（如「重刀法」= 气劲 + 根骨双前置）；默认单属性。
- **延伸**：社交技主属性 = 魅力、探索技主属性 = 福缘，沿用同一套规则（细节归各自文档）。

### 3.3 配置与数值

- **配置位置**：每门武功 / 每件装备在其**各自配置**里写明主属性要求，config-driven，与战斗文档一致。
- **属性值范围（工作假设）**：先按 **1–100** 讨论门槛数值；**真实范围与成长曲线归属成长系统文档**，本 spec 不定死。

## 四、属性成长（占位）

- 属性**可成长**（随等级 / 内容提升）。
- 成长来源、成长曲线、属性上限、属性点获取方式 → **成长系统文档**（待创建）。
- 本 spec 仅声明"属性可成长"，不定具体数值。

> 相关：属性成长是**才华获取**的一种触发条件；但才华是独立的触发式、多渠道系统，**不在本文档成文**，详见才华系统文档。

## 六、关键决策与理由（记录）

1. **属性退出骰池**：让"骰子从哪来"（武功 / 装备 / 天赋 / 才华）与"骰子多强"（属性缩放）职责分离；属性不再和骰面 / 命中纠缠，避免与"命中是骰面结果"的机制冲突。
2. **气劲 / 内息分立**：让外家 / 内家在属性层就有差异，支撑流派构建。
3. **悟性 / 精准只描述为"X 类武学修正"**：不在属性层绑定具体效果（暴击 / 穿甲等），把这些实现细节留给武功 / 公式文档，保持属性层稳定。
4. **门槛统一走"主属性"**：所有属性行为一致，悟性不再特殊化；武功硬卡、装备软惩，兼顾目标感与体验。
5. **空手保底「挥击骰体」**：确保角色在任何状态下都保有最低限度的行动能力。
