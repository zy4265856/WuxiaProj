# 2D武侠游戏设计大纲

## 一、游戏概览

### 游戏定位
一款2D武侠题材的回合制策略RPG，融合开放世界探索与深度骰子战斗机制。玩家在江湖世界中扮演武者，通过修炼武功、收集装备、结交NPC来成长，最终达成个人目标。

### 核心体验循环
```
探索世界 → 遭遇事件/战斗 → 骰子结算 → 获得成长 → 继续探索
```

**探索阶段**：玩家在多个互联的地图区域中自由探索，与NPC对话、接受任务、发现秘密。每个区域有独特的氛围、势力和事件。

**战斗阶段**：遭遇敌人时进入回合制战棋战斗。玩家每回合投掷骰子，根据骰子类型决定可用行动，结合走位和策略击败敌人。

**结算阶段**：骰子结果结合属性、装备、武功产生最终效果。战斗胜利获得经验、装备、武功秘籍等；剧情事件根据骰子结果产生不同分支。

### 独特卖点
- **骰子驱动的武侠战斗**：告别传统回合制的固定伤害，每次投掷都充满变数与惊喜
- **灵活的骰子配置**：玩家可自由搭配武功与装备，构建独特的战斗风格
- **策略与运气并重**：同样配置下，骰子结果的随机性要求玩家灵活应变

## 二、核心系统 — 骰子机制

### 骰子堆叠模型
任何行动的骰子池由两部分组成：

```
行动骰子池 = 基础骰 + 联动骰
```

> **骰池来源**：武功 + 装备 + 天赋 + 才华（+ 成长奖励的自定义骰面）。同一来源可同时提供基础骰与联动骰。
> **属性不产出骰子**：属性只做「骰面公式缩放」与「武功 / 装备门槛」，详见 [属性系统设计](2026-06-25-attribute-system-design.md)。

**基础骰**：提供主要行动的骰子（攻击骰、移动骰、休息骰等）。
- **保底约定**：空手且无任何武学时，角色仍持有一个预设的**「挥击骰体」**（保底攻击骰），确保任何时候都可行动。

**联动骰**：需与基础骰配合使用的骰子，按来源分类：

- **装备**：武器、防具分配到相关行动
- **携带道具**：特定物品提供常驻或条件触发骰子
- **武功**：学习武功后获得武功骰，并激活匹配装备的骰子
- **天赋**：角色创建时固有的特殊能力，无法在游戏内被修改
- **才华**：角色在游戏进程中可以获得的特殊能力
- **队友羁绊**：特定队友组合提供协同骰子

联动系统设计为可扩展架构，后续可继续添加新的联动来源。

### 骰子类型

**数字骰**：基础骰子类型，直接产生数值结果。常见面数包括D4、D6、D8、D10、D12，高级内容提供更大面数。

**效果骰**：特殊骰子，部分面数为特殊效果：
- 武学类：暴击、连击、破防、流血等
- 社交类：威慑、魅力、说服等
- 探索类：洞察、幸运、发现等
- 各门派/类型可能有独特效果面

### 结算框架（待扩展）
不同行动类型有各自的结算逻辑：
- **战斗行动**：基于骰子结果计算伤害、防御等
- **社交互动**：基于骰子结果影响NPC好感度、说服成功率等
- **探索事件**：基于骰子结果触发发现、避免危险等

## 三、角色成长系统

### 成长维度
角色主要通过三个维度获得成长：

**属性成长**：角色拥有十维属性（气劲 / 内息 / 根骨 / 悟性 / 身法 / 定力 / 福缘 / 魅力 / 精力 / 精准），代表其固有能力水平。属性随等级提升，作为**骰面公式的缩放输入**，并作为**武功学习 / 装备使用的主属性门槛**——但**属性不产出骰子**。详见 [属性系统设计](2026-06-25-attribute-system-design.md)。

**武功习得**：武功是角色成长的核心途径。学习武功后获得武功骰子，并激活匹配类型装备的联动骰。武功按功能划分为不同类别，每类对应不同的行动类型和装备需求。武功可通过拜师、秘籍、奇遇等多种途径获得。

**等级进阶**：角色通过参与游戏内容获得经验并升级。升级时提供属性增长和武功学习资格，可能触发特殊成长事件。

### 成长来源
- 战斗胜利
- 完成任务
- 探索发现
- 社交互动
- 特殊事件

## 四、战斗系统

> **详细设计文档**：战斗系统的详细设计见 [战斗系统详细设计](2026-06-15-combat-system-detailed-design.md)

### 战场结构
网格化的2D战棋地图，每个单位占据一格。地图包含障碍物、地形效果、互动元素。

### 回合结构
每回合开始时，系统投掷角色所有战斗关联骰子（基础骰 + 联动骰），形成本回合的"骰子手牌"。

### 骰子类型与效果
**骰子的功能组成**
- 骰面：一枚骰子由多个骰面组成，每一个骰面有一个或多个行动效果和开销
  - 行动效果：骰面被骰出使用的效果
  - 开销：执行骰面的开销，通常包含行动力开销和其他数值开销，部分特殊情况可能还包含物品开销
- 骰体效果：骰子本身的效果
  - 入场效果：进入战场后本场战斗内持续提供的效果
  - 开始效果：单位行动回合开始时，如若骰子仍然可用，则立刻生效的效果
  - 留用效果：单位行动回合结束时，如果骰子未被打出使用，则立刻生效的效果

**基础骰**：
- **攻击骰**：提供攻击行动，数值影响伤害
- **移动骰**：提供移动能力，骰面效果数值决定本次行动前的移动格数上限
- **休息骰**：恢复气血/内力

**联动骰**：需与基础骰配合使用
- **武器骰**：改变攻击骰的攻击类型（切割、穿刺、钝击等）
- **武功骰**：与匹配的攻击骰类型联动，扩展效果和范围

### 行动力系统
角色拥有"行动力"资源，使用骰子消耗行动力，部分骰子（如休息骰）可回复行动力。

### 行动执行流程
0. **进入战斗（仅一次）**：获得所有骰子的入场效果
1. **投骰阶段**：掷出所有骰池中的骰子，并执行骰子开始效果
2. **移动阶段**：打出移动骰进行移动（可回溯调整）
3. **行动阶段**：打出骰子与对应开销执行骰子行动效果
   - 联动骰必须跟随对应的基础骰一起打出
   - 武器骰改变攻击类型
   - 武功骰扩展范围和效果
4. **结束阶段**：剩余骰子执行骰子留手效果，回合结束，所有可用骰子回池

### 范围与组合
骰子组合决定攻击范围：例如
- 单一剑攻击骰：4向直线2格
- 剑攻击骰 + 剑法骰：额外增加斜角2格
- 移动骰替代消耗：消耗移动骰增加伤害或位移效果

### 胜负条件与奖励
- 胜利：配置驱动的胜利条件，通常包含击败特定敌人/击败本场全部敌方单位/任意人员存活/指定人员存活/坚持一定时间周期
- 失败：角色生命归零
- 奖励：经验、装备、武功秘籍、消耗品

## 五、世界与探索

> **设定总纲**：世界的基调（中武偏写实）、势力格局、核心冲突、玩家定位等顶层设定见 [武侠世界观总纲](2026-06-26-worldview-design.md)；本节描述其下的探索玩法结构。

### 世界结构
游戏世界由多个互联的地图区域组成，每个区域是独立的场景，包含：
- **地理环境**：山脉、城镇、竹林、江河等
- **势力分布**：不同门派、帮派的控制区域
- **NPC社群**：居民、商人、武者、门派弟子等
- **探索点**：隐藏地点、秘境、宝箱等

### 地图交互
玩家在世界地图中可进行多种交互：
- **自由移动**：在区域间或区域内移动
- **NPC对话**：与角色交流，获取信息、接受任务、建立关系
- **环境互动**：调查物体、触发机关、发现秘密
- **随机遭遇**：行走时可能触发敌人或事件

### NPC系统
世界中有大量NPC角色，每个角色有其属性、立场、故事。玩家可通过骰子判定影响与NPC的关系：
- **社交互动**：对话选项需要骰子判定成功与否
- **好感系统**：互动结果影响NPC态度
- **势力关系**：行为影响与各势力的关系
- **羁绊系统**：特定NPC可成为队友，提供联动骰

### 事件系统
世界探索中会遇到各种事件，使用骰子判定结果：
- **战斗事件**：遭遇敌人、强盗、野兽等
- **剧情事件**：奇遇、谜题、道德抉择等
- **发现事件**：找到秘籍、宝物、隐藏地点等
- **社交事件**：路见不平、酒馆传闻等

## 六、物品与消耗品

### 物品分类
游戏中的物品主要分为以下类型：
- **装备**：武器、防具、饰品等，可分配到行动中提供联动骰
- **武功秘籍**：学习新武功的载体
- **消耗品**：战斗内外使用的一次性物品
- **任务物品**：剧情相关物品
- **珍稀材料**：用于打造或交易的资源

### 装备系统
装备可分配到特定行动类型，提供联动骰：
- **武器**：改变攻击骰的攻击类型，提供额外骰子
- **防具**：提供防御相关骰子
- **饰品**：提供特殊效果骰子或属性加成
- 装备分配可自由调整，适应不同战斗需求

### 消耗品
消耗品提供即时的数值修改或buff效果：
- **回复类**：恢复气血、内力
- **增益类**：临时提升属性、增加骰子数量
- **特殊类**：解毒、复活、状态解除等

### Buff系统
Buff基于存在周期为单位提供增益效果：
- **回合类**：持续若干战斗回合
- **时间类**：持续若干游戏时间
- **永久类**：永久生效（特殊获得）
- Buff效果包括：临时数值增加、特定功能骰子数量增加、特殊效果触发等

## 七、技术架构（简述）

### 数据模型核心
游戏围绕"骰子堆叠"模型设计数据结构：

**骰子数据**：
- 骰子类型、面数、效果面定义
- 骰子来源标记（武功/装备/天赋/才华等）

**行动数据**：
- 行动类型定义（攻击/移动/休息等）
- 行动允许的骰子类型匹配规则
- 行动的骰子池计算逻辑

**联动系统数据**：
- 各联动来源的定义（装备/武功/天赋等）
- 匹配规则（哪些联动骰可与哪些基础骰配合）
- 效果叠加规则

### 扩展性设计
- **联动系统可扩展**：新联动来源（如阵法、环境）可无缝接入
- **骰子类型可扩展**：新骰子类型和效果面可添加
- **行动类型可扩展**：新行动类型可定义
- **事件系统可扩展**：新事件类型和判定逻辑可添加

### 存档系统
- 角色状态（属性、等级、已学武功）
- 世界状态（已探索区域、NPC关系、任务进度）
- 物品栏（装备、消耗品、任务物品）

---

## 结语

本大纲确立了2D武侠游戏的核心框架：**骰子驱动的战棋战斗 + 开放世界探索**。核心创新点在于：

1. **骰子堆叠模型**提供灵活的养成深度
2. **骰子决定可用行动**形成独特策略体验
3. **联动系统可扩展**为后续内容预留空间

各系统的详细设计将在后续阶段逐步完善。
